you will need:

1- Mac or Hackintosh with OSX preinstalled (you can use vmware installation if dont have preinstalled)

2- Developer Preview Application which available free if you are developer or download at any torrent.
3- Latest version of MacPwn for Yosemite Developer Preview. here
4- Pendrive 8GB or more.

Rename apllication to "Install OS X 10.10 Developer Preview.app" if the name is different and drag it to the Application folder as shown in picture below.

Insert your pendrive and open disk utility app, select the USB disk and then click partition on the right panel.

select partition layout to 1 partition and select option and choose GUID Partition table

rename the usb disk to MACPWN format MAC OSX extended (Journaled)

click apply > partition

next, open MACPWN and click continue


click change install location

select MACPWN as install destination > continue

click customize

for people that has a same laptop as me, you can tick like in the picture here.

i use clover instead of chameleon so chameleon configuration is not needed and kernel and MBR Patch is based on your installer, for example here mine is DP1

and click install, it sure take some time to finish, just leave it up.
